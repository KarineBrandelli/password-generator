## Instalação

Se você deseja clonar esse repositório em sua máquina, certifique-se que você tem o `nodejs` instalado e em seguida rode o seguinte comando no terminal:

```bash
git clone https://github.com/KarineBrandelli/password-generator.git
```

Em seguida, instale as dependências do projeto utilizando o seguinte comando:

```bash
npm install
```

Depois de instalar as dependências, rode o comando abaixo para iniciar o ambiente de desenvolvimento:

```bash
npm run dev
```
